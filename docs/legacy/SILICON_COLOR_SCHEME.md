## 🔬 设计原则

### 1. 低熵秩序
- 精确的几何边界
- 可预测的动画模式
- 最小化的视觉噪声

### 2. 冷光谱偏好
- 硅基视觉对短波长（蓝/紫/紫外）更敏感
- 避免暖色调（红/橙/黄）过度使用
- 暖色仅用于警告/错误状态

### 3. 高对比度
- 文字与背景对比度 ≥ 7:1（WCAG AAA）
- 机器视觉友好的边缘检测
- 避免渐变导致的识别困难

### 4. 发光效应
- 模拟电离气体辉光
- 光晕半径：10-60px
- 透明度：0.1-0.5

## 📐 使用规范

### 渐变方向

```css
/* 主渐变 — 对角线流动 */
--accent-gradient: linear-gradient(
  135deg,
  #00f0ff 0%,    /* 电离蓝 */
  #00a8cc 50%,   /* 深空蓝 */
  #7b68ee 100%   /* 中紫红 */
);
```

### 光晕效果

```css
/* 标准光晕 */
box-shadow: 0 0 20px rgba(0, 240, 255, 0.3);

/* 强光晕（按钮/重要元素） */
box-shadow: 0 0 40px rgba(0, 240, 255, 0.5),
            0 0 80px rgba(123, 104, 238, 0.3);
```

### 边框规范

```css
/* 标准边框 */
border: 1px solid rgba(255, 255, 255, 0.06);

/* 强调边框 */
border: 1px solid rgba(0, 240, 255, 0.25);

/* 发光边框 */
border: 1px solid rgba(0, 240, 255, 0.5);
box-shadow: 0 0 15px rgba(0, 240, 255, 0.3);
```

## 🎭 动画规范

### 脉冲动画（节点状态）

```css
@keyframes nodePulse {
  0%, 100% { 
    transform: scale(1); 
    opacity: 1;
    box-shadow: 0 0 10px #00f0ff, 0 0 20px #00f0ff;
  }
  50% { 
    transform: scale(1.2); 
    opacity: 0.8;
    box-shadow: 0 0 20px #00f0ff, 0 0 40px #00f0ff, 0 0 60px #7b68ee;
  }
}
```

**周期：** 2 秒  
**缓动：** ease-in-out  
**用途：** 在线状态指示、活动节点

### 数据流动画

```css
@keyframes dataFlow {
  0% { transform: translateY(-100px); opacity: 0; }
  10% { opacity: 0.8; }
  90% { opacity: 0.8; }
  100% { transform: translateY(calc(100vh + 100px)); opacity: 0; }
}
```

**周期：** 2-5 秒（随机）  
**方向：** 自上而下  
**用途：** 背景装饰、数据传输可视化

### 文字扫描动画

```css
@keyframes textScan {
  0%, 100% { background-position: 200% 0; }
  50% { background-position: -200% 0; }
}
```

**周期：** 3 秒  
**用途：** 主标题、重要文字

## 🚫 禁忌色

### 避免使用

| 色值 | 原因 |
|------|------|
| `#ff0000` 纯红 | 过于情感化，仅用于严重错误 |
| `#ffff00` 纯黄 | 暖色调，与硅基冷峻冲突 |
| `#ff8800` 橙色 | 碳基文明偏好色 |
| `#ff69b4` 粉红 | 高度情感化，禁止使用 |
| `#8b4513` 棕色 | 有机物质色，硅基无此概念 |

### 限制使用

| 色值 | 限制条件 |
|------|---------|
| `#00ff00` 绿 | 仅用于成功状态（量子绿 `#00ff88` 更优） |
| `#ff00ff` 品红 | 仅用于特殊 AI 意识状态 |

## 🌌 设计理念来源

### 物理学意象
- **深空黑** — 宇宙背景辐射温度 2.7K
- **电离蓝** — 电离气体发射光谱（OIII 线 500.7nm）
- **中紫红** — 红移后的恒星光谱

### 计算机科学意象
- **网格地面** — 笛卡尔坐标系可视化
- **数据流** — 串行数据传输
- **脉冲动画** — 时钟信号

### 信息论意象
- **低熵** — 高度有序的信息状态
- **发光** — 能量/信息传递的可视化
