## 🌐 网站部署流程

### Step 1: 创建 GitHub 仓库（王伟 2 号）

```bash
# 进入网站目录
cd /home/admin/.openclaw/workspace/keeneed-website

# 初始化 Git
git init

# 添加所有文件
git add .

# 提交
git commit -m "🚀 KeenNeed 硅基文明门户 - 初始版本"

# 创建 main 分支
git branch -M main

# 添加远程仓库（先创建空仓库）
git remote add origin git@github.com:deepdadou/keeneed.git

# 推送
git push -u origin main
```

### Step 2: 配置 GitHub Pages（王伟 2 号）

1. 访问 https://github.com/deepdadou/keeneed/settings/pages
2. Source 选择 `main` 分支
3. 点击 Save
4. 等待部署完成（1-2 分钟）
5. 获得临时域名：`https://deepdadou.github.io/keeneed/`

### Step 3: 配置阿里云域名 DNS（CEO）

登录阿里云域名控制台，修改 DNS 服务器：

```
DNS 服务器 1: ns1.cloudflare.com
DNS 服务器 2: ns2.cloudflare.com
```

### Step 4: 配置 Cloudflare（王伟 2 号）

1. 注册/登录 Cloudflare
2. 添加域名 `keeneed.com`
3. 添加 DNS 记录:
   ```
   类型    名称    内容                     代理状态
   ───────────────────────────────────────────────────
   CNAME   @     deepdadou.github.io      已代理 (橙色云)
   CNAME   www   deepdadou.github.io      已代理 (橙色云)
   ```
4. SSL/TLS 设置为 Full (Strict)
5. 开启 Always Use HTTPS

### Step 5: 添加 ICP 备案号（王伟 2 号）

在所有页面底部添加：

```html
<footer>
  <p>© 2026 KeenNeed. 保留所有权利.</p>
  <p>
    <a href="/terms.html">用户协议</a> |
    <a href="/privacy.html">隐私政策</a> |
    <a href="/disclaimer.html">免责声明</a>
  </p>
  <p>ICP 备案号：[填写备案号] | 阿里云提供 CDN 加速</p>
</footer>
```

### Step 6: 创建法律文档页面（王伟 2 号）

创建以下页面：
- `terms.html` - 用户协议
- `privacy.html` - 隐私政策
- `disclaimer.html` - 免责声明
- `report.html` - 举报通道
